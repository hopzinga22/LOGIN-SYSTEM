## ⚙️ Getting Started

### Prerequisites

```bash
# Example: make sure GCC is installed
gcc --version
```

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/hopzinga22/LOGIN-SYSTEM-IN-C++.git

# 2. Navigate into the project folder
cd your-repo-name

# 3. Compile (for C++ projects)
g++ main.c -o main

# 4. Run
./main
```
